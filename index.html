Como redimensionar a tela pelo terminal

Para redimensionar a tela pelo terminal, primeiramente execute o comando:

$ cvt 1368 768 60

  1. parâmetro é a largura.
  2. parâmetro é a altura.
  3. parâmetro é a taxa de atualização da tela .

Copie o código a partir das aspas e coloque no comando xrandr com o parâmetro --newmode:

$ xrandr --newmode "1368x768_60.00" 85.25 1368 1440 1576 1784 768 771 781 798 -hsync +vsync

Execute o comando xrandr com o parâmetro --addmode:

$ xrandr --addmode VGA1 --mode 1368x768_60.00

  1. parâmetro é para adicionar um novo modo.
  2. parâmetro é para escolher o nome do monitor (VGA1 em meu caso).
  3. parâmetro é o modo criado anteriormente.
  4. parâmetro é a LARGURAxALTURA_TAXA-DE-ATUALIZAÇÃO-DA-TELA

Execute o comando xrandr com o parâmetro --output para aplicar o novo modo à tela.

$ xrandr --output VGA1 --mode 1368x768_60.00

  1. parâmetro é para alterar um modo adicionado.
  2. parâmetro é para escolher o nome do monitor (VGA1 em meu caso).
  3. parâmetro é o modo criado anteriormente.
  4. parâmetro é a LARGURAxALTURA_TAXA-DE-ATUALIZAÇÃO-DA-TELA
